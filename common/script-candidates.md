# Script Candidates

将来的にスクリプト化できそうな作業の候補です。初期運用では手動管理を優先し、繰り返しが増えた段階で実装します。

## 優先度 高

- `video-index.md` から動画一覧テーブルを検査する。
- サムネイル画像の比率をチェックする。
- VID番号の重複と欠番をチェックする。
- テンプレートから新規動画メモを作る。
- タイトル、説明文、サムネイル文字が英語になっているか簡易チェックする。

## 優先度 中

- 説明文テンプレートにテーマ語彙を差し込む。
- YouTube向け説明文のSEOキーワード出現を確認する。
- サムネイルレビューをCSVまたはJSONに変換する。
- Git上の動画情報からNotion転記用のMarkdownを生成する。

## 優先度 低

- 公開済み動画のYouTube URLを検査する。
- サムネイル候補画像のファイル名をVID番号に合わせてリネームする。
- 過去の改善メモから次回プロンプト候補を自動生成する。
