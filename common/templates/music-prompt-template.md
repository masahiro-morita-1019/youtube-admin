# Music Prompt Template

音楽生成プロンプトのテンプレートです。ツールごとの制約に合わせて短く調整して使います。

## Japanese Base

```text
{Theme} をイメージした {Mood} 向けのアンビエント音楽。
柔らかい空間音、ゆっくり変化するパッド、静かなシネマティックアンビエンス、深く落ち着く雰囲気、ミニマルなリズム、没入感のある音響。
睡眠・リラックス・集中・瞑想に適したサウンド。
ボーカルなし。
```

## English Base

```text
Ambient music inspired by {theme}, designed for {use_case}.
Soft spatial textures, slowly evolving pads, quiet cinematic ambience, deep calming atmosphere, minimal rhythm, immersive atmospheric soundscape, no vocals.
```
